NLP + Latent Dirichlet Allocation (LDA)

Sage City forum posts for the last two years were reviewed using sentiment analysis along with LDA topic extraction, revealing data trends at specific dates. Sentiment over time is plotted by various users against known time markers (R&D release cycles) at these interesting data events. LDA scoring technique chosen was the C_umass coherence measure.


Processing Steps Taken

Using a Jupyter Notebook the following steps were taken:

  1. Data procurement (Sage City Forum Management)
  2. Data Cleaning: html tag removal, Lemmatization
  3. Data Preparation: The use of bi and tri-grams
  4. Feature Engineering: Building features used for time series analysis, removing html tags
  5. Fine tuning (iterative process): Tuning the number of topics, adding stop words, setting the number of words and passes for the LDA model to execute.
  6. Human Review
